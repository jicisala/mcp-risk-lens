# Professional MCP and AI Agent review

MCP Risk Lens covers deterministic configuration checks. Production systems also need review at
the trust-boundary, authorization, runtime, and operational layers.

## Early-adopter packages

### MCP readiness review — USD 700

Best for a team preparing its first internal or customer-facing MCP deployment.

- Up to 10 MCP servers or equivalent tool integrations
- Static configuration review with credential-safe evidence
- One 60-minute architecture interview
- Trust-boundary and high-risk action map
- Prioritized findings and remediation plan
- Five-business-day target delivery after inputs are accepted

### Agent governance sprint — from USD 2,000

Best for production agents that can write data, call business APIs, or trigger irreversible actions.

- Everything in the readiness review
- Authorization and least-privilege design
- Human-approval and high-risk action policy
- Prompt-injection and tool-misuse test plan
- Audit logging and OpenTelemetry coverage review
- Regression checklist and implementation backlog
- Two-week target delivery after scope confirmation

Prices are early-adopter targets, not an automatic acceptance of work. Final scope, availability,
currency, taxes, payment milestones, and delivery dates must be confirmed in writing.

## Safe intake

An initial inquiry only needs:

1. Public company or project description
2. Approximate number of MCP servers and tools
3. Whether tools are read-only, write-capable, or financially sensitive
4. Desired outcome and target date

Do **not** post secrets, proprietary configuration, customer data, source code, private endpoints,
or non-public architecture in GitHub issues. Sensitive material is accepted only after scope,
confidentiality, and a secure transfer method are agreed.

## Exclusions

- No unauthorized security testing
- No credential handling through public channels
- No compliance certification or legal opinion
- No guarantee that static review discovers every vulnerability
