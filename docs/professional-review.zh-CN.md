## 安全的首次咨询

首次只需要提供：

1. 可公开的企业或项目简介
2. MCP Server 与工具的大致数量
3. 工具属于只读、可写还是涉及资金操作
4. 希望解决的问题与目标时间

请勿在 GitHub Issue 中提交密钥、私有配置、客户数据、源代码、私有地址或非公开架构。敏感材料
只会在范围、保密义务和安全传输方式确认后接收。
