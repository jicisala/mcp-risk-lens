# Security policy

MCP Risk Lens analyzes configuration text and never starts configured servers. Please do not
attach live credentials to public security reports or issues.

For a suspected vulnerability in the scanner, open a private GitHub security advisory after the
repository is published. Include a minimal synthetic reproduction and avoid real secrets or
customer data.

This project is an early static-analysis tool. Its output is not a penetration test, formal risk
acceptance, or compliance certification.
